# chess-in-unity
A simple but beautiful chess game in Unity.

## How to set up:
- **Install Unity 6.3 LTS**: Make sure you have the exact version to avoid compatibility issues.
- **Install GitHub Desktop**: The easiest way to manage the project.
- **Install Git LFS**:
``
git lfs install
git lfs pull
``
- **Clone this repo**
- **Open via Unity Hub**: Click "Add" -> "Add project from disk" and select the folder.
- Make sure the "**Version Control**" Setting in Unity is set to "**Visible Meta Files**"

# WICHTIG!
Arbeitet niemals in unterschiedlichen Branches gleichzeitig an der gleichen Szene (.unity-Datei). Unity-Szenen lassen sich technisch nicht sauber zusammenführen (mergen). Wenn zwei Personen gleichzeitig Änderungen an derselben Szene hochladen, wird die Datei beim Mergen unbrauchbar und die Arbeit geht verloren.

Never modify the same scene (.unity file) in different branches at the same time. Unity scenes cannot be merged effectively. Attempting to merge two different versions of the same scene will almost certainly corrupt the file and lead to a total loss of work.
